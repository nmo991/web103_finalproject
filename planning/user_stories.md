# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

1. **Skill Learner**: A user seeking to acquire a new skill from community members. They browse available skills, initiate exchanges, and leave reviews after learning.

2. **Skill Teacher**: A user offering their expertise in one or more skills and willing to share knowledge with learners. They manage incoming exchange requests and track their teaching history.

3. **Community Member**: A user who both teaches and learns skills. This role encompasses users who actively participate in the skill-sharing ecosystem bidirectionally.

## Draft User Stories

1. As a Skill Learner, I want to browse all available skills on the platform, so that I can discover what the community has to offer.

2. As a Skill Learner, I want to filter skills by category (e.g., languages, instruments, fitness), so that I can find skills relevant to my interests.

3. As a Skill Learner, I want to view detailed information about a skill including the teacher's profile and expertise level, so that I can decide if it's the right match for me.

4. As a Skill Learner, I want to send a skill exchange request to a teacher without leaving the skill detail page, so that I can quickly initiate exchanges.

5. As a Skill Teacher, I want to view incoming skill exchange requests from learners, so that I can decide who to teach.

6. As a Skill Teacher, I want to accept or decline a skill exchange request with a message, so that the learner knows my decision and availability.

7. As a Community Member, I want to see all my active and completed skill exchanges in one place, so that I can track my learning and teaching progress.

8. As a Community Member, I want to leave a review and rating for someone I exchanged skills with, so that I can help other community members make informed decisions.

9. As a Community Member, I want to view my profile showing the skills I teach and want to learn, so that others can find me as a potential exchange partner.

10. As a Skill Learner, I want to search for specific skills by name (e.g., "Spanish" or "guitar"), so that I can quickly find exactly what I'm looking for.

11. As a Community Member, I want to see my average rating and review count on my profile, so that I can understand how the community perceives my exchanges.

12. As a Skill Teacher, I want to edit or remove skills I'm offering from my profile, so that my offerings stay current with what I'm actually teaching.
